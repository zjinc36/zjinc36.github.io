# Maven查看_依赖jar包_获取版版本号 _license

直接在命令行运行

```bash
mvn project-info-reports:dependencies
```

![](../images/2024/08/20240823152555.png)

在生成的tar目录下 找到网址，直接打开

![](../images/2024/08/20240823152616.png)
