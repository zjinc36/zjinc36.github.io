git所拥有的”区域”

  • Workspace：工作区
  • Index / Stage：暂存区
  • Repository：仓库区（或本地仓库）
  • Remote：远程仓库

git忽略文件

从未提交过的文件可以用.gitignore

  • 添加到暂存区，但从来没有提交到本地仓库的文件
  • 也就是添加（add）之后从来没有提交（commit）过的文件，可以使用.gitignore忽略该文件
  • 该文件只能作用于未跟踪的文件（Untracked Files），也就是那些从来没有被 git 记录过的文件
