#	Eclipse显示隐藏文件
点号.开头的文件是隐藏文件，很多时候我们需要编辑它，比如.gitignore文件、.htaccess文件，在Eclipse作下设置，就不用麻烦地去另外打开编辑它了。
方法是：
1.	在Navigation或Project视图下
2.	点右上角的三角形图标
3.	弹出的菜单中点filter
4.	然后取消`.* resources`一项
5.	保存就可以了
