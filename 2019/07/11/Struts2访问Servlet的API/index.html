Servlet 和 Action 的区别

参考:https://blog.csdn.net/siwuxie095/article/details/77075528
Servlet：默认在第一次访问时创建，且只创建一次，是单实例对象
Action：访问时创建，且每次访问都会创建，创建多次，是多实例对象
