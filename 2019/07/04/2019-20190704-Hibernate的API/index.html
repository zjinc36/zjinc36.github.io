SessionFactory

简介

SessionFactory内部维护了Hibernate的连接池和Hibernate的二级缓存(不讲)
是线程安全的对象
一个项目创建一个对象即可

Session

简介

Session代表的是Hibernate与数据库的连接对象,即与数据库交互桥梁
不是线程安全
