##  安装notepadqq
### 作用
Windows 上的 Notepad++，开发者们已经将它移植到 Linux，名为 Notepadqq
它是一个简单而强大的文本编辑器，你可以在日常使用它，或用于各种语言进行编程
### 安装
```
$ sudo add-apt-repository ppa:notpadqq-team/notepadqq
$ sudo apt-get update
$ sudo apt-get install notepadqq
```
