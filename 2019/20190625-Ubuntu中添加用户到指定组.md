_假定用户名为`user1`,组名为`groupA`_
##  查看用户具体所在的组

```
    groups user1
```

##  添加用户到指定用户组
```
    usermod -a -G groupA user1

    注意:
    一定要使用-a参数(代表append)
    如果没有使用该参数,会使你离开其他用户组，仅仅做为用户组groupA的成员
```
