# 配置命令提示符颜色

---

## 如果不存在则创建~/.bash_profile文件

```bash
vim ~/.bash_profile
```

键入如下内容

```bash
# if running bash  
if [ -n "$BASH_VERSION" ]; then
    # include .bashrc if it exists  
    if [ -f "$HOME/.bashrc" ]; then
        . "$HOME/.bashrc"
    fi
fi
```

## 在~/.bashrc文件中添加如下内容

```bash
vim ~/.bashrc
```

键入如下内容

```bash
export PS1='\[\e[1;33m\]\u@\h:\[\e[0m\]\[\e[1;36m\]\w\[\e[1;33m\]\[\e[0m\]\[\e[1;36m\]\$ \[\e[0m\]'
```

立即生效

```bash
source ~/.bashrc
```
