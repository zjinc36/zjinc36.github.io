标识文件

引入名词Blob

  • Blob的定义
    文件名为内容的SHA-1值的文件,具体来说,如下图,

  • 更具体一点来说文件名 == SHA-1(类型 + 内容长度 + \0 + 内容 )

    其中type用以表示该Blob文件的类型,之所以要区分类型,是因为Blob文件根据内容的不同,又可以分为Tree Object,Commit Object等,用以不同的作用

用Blob标识文件

每个Git管理的文件,在.git录下,都有一个文件与之对应,而这个与之对应的文件称为Blob,同时这个与之对应的文件的文件名为内容的SHA-1值
