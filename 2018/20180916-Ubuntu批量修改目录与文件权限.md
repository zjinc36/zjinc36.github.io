##  问题
+   家目录下,所有目录的"可运行"权限全部消失,但文件的权限还是正常的
+   现在要给目录重新赋予"可运行"权限,但又不能给文件"可运行"权限
##  修改一个目录下所有文件权限
```
    chmod -R 0755 <要修改的目录>
```

##  批量修改一个目录下所有文件的权限
```
    find /usr/local -type d -exec chmod 0755 {} \;
```
